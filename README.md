# The Dat Protocol
This repository contains the source for "The Dat Protocol" - an implementers
guide.

## Requirements
Building the book requires [mdbook](https://github.com/azerupi/mdBook).
```sh
$ cargo install mdbook
```

## Building
To build the book run
```sh
$ mdbook build
```
